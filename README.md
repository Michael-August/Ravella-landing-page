# Ravella Design Hub

## Project info

Ravella Design Hub is a modern web application promoting wellness and sustainable income opportunities through a community-driven platform.

## Getting Started

The project uses Node.js and npm. Make sure you have them installed:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd ravella-landing-page

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Next.js 14
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- PostCSS

## Development Workflow

**Local Development:**
- Clone the repository and make changes locally
- Test your changes with `npm run dev`
- Commit and push your changes to the repository

**Edit Files:**
- Use your preferred IDE to edit files locally
- Or edit directly in GitHub using the pencil icon
- GitHub Codespaces is also available for cloud-based development

## Deployment

To deploy this project:

1. Build the project: `npm run build`
2. Start the production server: `npm start`
3. Deploy using your preferred hosting platform
